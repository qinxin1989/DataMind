## 系统简介

### 什么是模块化架构系统？

模块化架构系统是一个灵活、高效、安全的企业级管理平台。通过模块化设计，您可以：

- ✅ **按需启用功能** - 只启用需要的模块，简化界面
- ✅ **灵活扩展** - 随时安装新模块，扩展功能
- ✅ **独立管理** - 每个模块独立配置，互不干扰
- ✅ **安全可靠** - 完善的权限控制和数据隔离

### 系统特点

| 特点 | 说明 |
|------|------|
| 🚀 **高性能** | API响应 < 5ms，并发支持 2857 QPS |
| 🔒 **高安全** | 安全评分 95/100，OWASP 合规 90% |
| 🎯 **易使用** | 直观的界面，简单的操作 |
| 🔧 **易维护** | 模块化设计，便于升级和维护 |

### 系统架构

```
用户界面
    ↓
模块管理系统
    ↓
功能模块 (18个)
    ↓
核心服务
    ↓
数据库
```

## 快速开始

### 第一步：登录系统

1. 打开浏览器（推荐 Chrome、Firefox、Edge）
2. 访问系统地址：`http://[服务器地址]:3000`
3. 输入用户名和密码
4. 点击"登录"按钮

**测试账号**:
- 管理员: `uat_admin` / [密码]
- 普通用户: `uat_user` / [密码]

### 第二步：了解界面

登录后，您会看到：

```
┌─────────────────────────────────────┐
│  Logo    系统名称         用户 ▼   │  ← 顶部导航栏
├─────────────────────────────────────┤
│ 📁 菜单  │                          │
│ ├ 首页   │                          │
│ ├ 用户   │      主要内容区域        │  ← 主要工作区
│ ├ 模块   │                          │
│ └ ...    │                          │
└─────────────────────────────────────┘
```

**界面说明**:
- **顶部导航栏**: 显示系统名称、用户信息、退出按钮
- **左侧菜单**: 显示可用功能模块（根据权限自动过滤）
- **主要内容区**: 显示当前功能的操作界面

### 第三步：开始使用

点击左侧菜单中的任意功能，即可开始使用。

## 核心功能

### 1. 模块管理

**功能**: 管理系统功能模块

**位置**: 左侧菜单 → 模块管理

**操作步骤**:

#### 查看模块列表
1. 点击"模块管理"
2. 查看所有已安装模块
3. 查看模块状态（启用/禁用）

#### 启用/禁用模块
1. 找到要操作的模块
2. 点击"启用"或"禁用"按钮
3. 确认操作
4. 等待操作完成

#### 配置模块
1. 找到要配置的模块
2. 点击"配置"按钮
3. 修改配置参数
4. 点击"保存"

**提示**:
- ⚠️ 禁用模块后，相关功能将不可用
- ⚠️ 某些模块有依赖关系，禁用前请确认
- ✅ 配置修改后立即生效

---

### 2. 用户管理

**功能**: 管理系统用户

**位置**: 左侧菜单 → 用户管理

**操作步骤**:

#### 创建用户
1. 点击"新增用户"按钮
2. 填写用户信息：
   - 用户名（必填）
   - 密码（必填，至少8位）
   - 姓名（必填）
   - 邮箱（可选）
   - 电话（可选）
3. 选择角色
4. 点击"保存"

#### 编辑用户
1. 找到要编辑的用户
2. 点击"编辑"按钮
3. 修改用户信息
4. 点击"保存"

#### 删除用户
1. 找到要删除的用户
2. 点击"删除"按钮
3. 确认删除

**提示**:
- ⚠️ 用户名不能重复
- ⚠️ 密码必须包含字母和数字
- ⚠️ 删除用户后无法恢复

---

### 3. 角色管理

**功能**: 管理用户角色和权限

**位置**: 左侧菜单 → 角色管理

**操作步骤**:

#### 创建角色
1. 点击"新增角色"按钮
2. 填写角色信息：
   - 角色名称（必填）
   - 角色描述（可选）
3. 选择权限
4. 点击"保存"

#### 分配权限
1. 找到要配置的角色
2. 点击"权限"按钮
3. 勾选需要的权限
4. 点击"保存"

**提示**:
- ✅ 权限采用树形结构，勾选父节点会自动勾选子节点
- ✅ 权限修改后，该角色的所有用户立即生效

---

### 4. 菜单管理

**功能**: 管理系统菜单

**位置**: 左侧菜单 → 菜单管理

**操作步骤**:

#### 查看菜单
1. 点击"菜单管理"
2. 查看菜单树形结构
3. 展开/折叠菜单节点

#### 编辑菜单
1. 找到要编辑的菜单
2. 点击"编辑"按钮
3. 修改菜单信息
4. 点击"保存"

**提示**:
- ⚠️ 菜单最多支持3层
- ⚠️ 删除菜单前请确认没有子菜单

## 业务模块

### 1. AI 配置管理

**功能**: 配置和管理 AI 模型

**位置**: 左侧菜单 → AI 服务 → AI 配置

**操作步骤**:

#### 添加 AI 配置
1. 点击"新增配置"按钮
2. 选择 AI 类型：
   - OpenAI
   - 本地模型
   - 其他
3. 填写配置信息：
   - 配置名称
   - API 密钥
   - 模型名称
   - 其他参数
4. 点击"测试连接"验证配置
5. 点击"保存"

#### 查看 AI 统计
1. 点击"AI 统计"
2. 查看调用次数、成本等信息
3. 可按时间范围筛选
4. 可导出报表

**提示**:
- ✅ 支持多个 AI 配置
- ✅ 可以设置默认配置
- ⚠️ API 密钥请妥善保管

### 2. 数据采集中心

**功能**: 创建和管理网页爬虫

**位置**: 左侧菜单 → 数据采集中心

#### 2.1 爬虫管理

**创建爬虫任务**:
1. 点击"新增任务"按钮
2. 填写任务信息：
   - 任务名称
   - 目标 URL
   - 采集频率
3. 配置采集规则
4. 点击"保存"

**执行爬虫任务**:
1. 找到要执行的任务
2. 点击"执行"按钮
3. 等待执行完成
4. 查看采集结果

#### 2.2 采集模板配置

**配置采集模板**:
1. 点击"模板配置"
2. 输入目标 URL
3. 使用可视化工具选择元素：
   - 点击页面元素
   - 自动生成选择器
   - 预览采集结果
4. 保存模板

**使用 AI 爬虫助手**:
1. 点击"AI 助手"按钮
2. 输入目标 URL
3. AI 自动分析页面结构
4. 自动生成采集规则
5. 预览并调整
6. 保存配置

**提示**:
- ✅ AI 助手可以大大简化配置工作
- ✅ 支持分页采集
- ⚠️ 请遵守网站的 robots.txt 规则

### 3. AI 问答系统

**功能**: 基于文档的智能问答

**位置**: 左侧菜单 → AI 问答

#### 3.1 知识库管理

**创建知识库**:
1. 点击"新增知识库"按钮
2. 填写知识库信息：
   - 知识库名称
   - 描述
3. 点击"保存"

**上传文档**:
1. 进入知识库
2. 点击"上传文档"按钮
3. 选择文件（支持 PDF、Word、TXT）
4. 等待处理完成

#### 3.2 智能问答

**提问**:
1. 选择知识库
2. 在输入框输入问题
3. 点击"提问"按钮
4. 查看答案和引用来源

#### 3.3 数据源管理

**连接数据库**:
1. 点击"新增数据源"按钮
2. 选择数据库类型：
   - MySQL
   - PostgreSQL
   - SQL Server
3. 填写连接信息
4. 点击"测试连接"
5. 点击"保存"

**SQL 问答**:
1. 选择数据源
2. 用自然语言描述查询需求
3. AI 自动生成 SQL
4. 执行查询
5. 查看结果

**提示**:
- ✅ 支持多种文档格式
- ✅ 支持多个知识库
- ✅ 答案会标注引用来源

## 常见问题

### 登录问题

**Q: 忘记密码怎么办？**
A: 联系系统管理员重置密码。

**Q: 登录后看不到某些菜单？**
A: 这是正常的，菜单会根据您的权限自动过滤。如需访问某个功能，请联系管理员分配权限。

**Q: 登录后页面空白？**
A: 请清除浏览器缓存，然后重新登录。

### 使用问题

**Q: 如何启用一个模块？**
A: 进入"模块管理"，找到要启用的模块，点击"启用"按钮。

**Q: 禁用模块后数据会丢失吗？**
A: 不会，禁用模块只是隐藏功能，数据仍然保留。重新启用后数据会恢复。

**Q: 如何查看我的权限？**
A: 点击右上角的用户名，选择"个人信息"，可以查看您的角色和权限。

**Q: 页面加载很慢怎么办？**
A: 
1. 检查网络连接
2. 清除浏览器缓存
3. 尝试使用其他浏览器
4. 联系技术支持

**Q: 操作失败怎么办？**
A:
1. 查看错误提示信息
2. 检查输入是否正确
3. 检查是否有权限
4. 联系技术支持

### 功能问题

**Q: AI 配置测试连接失败？**
A: 
1. 检查 API 密钥是否正确
2. 检查网络连接
3. 检查 API 服务是否可用

**Q: 爬虫任务执行失败？**
A:
1. 检查目标 URL 是否可访问
2. 检查采集规则是否正确
3. 查看错误日志

**Q: 知识库文档上传失败？**
A:
1. 检查文件格式是否支持
2. 检查文件大小是否超限
3. 检查网络连接

**Q: 数据源连接失败？**
A:
1. 检查连接信息是否正确
2. 检查数据库是否可访问
3. 检查用户权限

### 性能问题

**Q: 系统响应慢？**
A: 系统已经过优化，API 响应时间 < 5ms。如果感觉慢，可能是：
1. 网络问题
2. 浏览器问题
3. 数据量过大

**Q: 文件上传慢？**
A: 文件上传速度取决于：
1. 文件大小
2. 网络速度
3. 服务器处理速度
