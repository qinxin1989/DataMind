# 更新日志

## [1.1.0] - 2025-01-12

### 新增功能
- ✨ **文件上传功能** - 前端支持直接上传 CSV/Excel/JSON 文件
  - 新增 `/api/upload` 端点支持文件上传
  - 自动检测文件类型
  - 支持最大 50MB 文件
  - 上传后自动创建数据源

- 📝 **测试套件** - 完整的 API 测试框架
  - 新增 `tests/run-tests.ts` 测试运行脚本
  - 新增 `tests/api.test.ts` 测试用例
  - 支持 13 个核心 API 端点测试
  - 添加 `npm run test` 命令

- 🔧 **后端改进**
  - 集成 multer 中间件处理文件上传
  - 自动创建 uploads 目录
  - 文件类型验证和大小限制

- 📱 **前端改进**
  - 数据源管理页面新增文件上传界面
  - 文件选择后自动检测类型
  - 显示文件大小和名称
  - 改进的表单交互体验

### 配置更新
- 更新 package.json 版本到 1.1.0
- 添加 multer 和 @types/multer 依赖
- 新增 test 脚本命令

### 文档更新
- 新增 CHANGELOG.md 版本记录
